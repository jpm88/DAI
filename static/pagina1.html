Diferencias de tratamiento de cadenas entre Python 2.7 y 3.1

 

Desde Python 3.x el tipo str contiene carácteres Unicode, y equivale al tipo unicode de Python 2.x.

 

Además el tipo binary de Python 3.x es el tipo str de Python 2.x

 

En Python 3.x para convertir pasar desde / a las distintas codificaciones están las funciones byte.decode() y str.encode()

 

En Python 2.x se pasa de str a unicode y viceversa con las funciones str.decode() y unicode.encode()
